Qu'est ce que le CSS ?

CSS est l'acronyme de Cascading Style Sheets ce qui signifie feuille de style en cascade.
Il permet de gérer la présentation d'une page Web comme la police, la couleur, les marges,image de fond etc...
Le but du CSS est de separer la structure d'un document HTML et sa presentation.

Exemples de balises:

Balises pour le texte

  • font-family

    • Permet d'appliquer une police à un élément textuel

  • font-size

    • Permet de changer la taille de sa police

  • text-align

    • Permet d'aligner son texte à droite, gauche ,centré ou justifié

  • color

    • Permet de changer la couleur du texte

    Balises pour le visuel

    • background-color

      • Permet de changer la couleur de l'arrière plan

    • background-image

      • Permet de mettre une image en arrière plan

    • background-repeat

      • L'image de fond est répété en mosaïque

    • background-position

      • Permet de positionner l'image de fond

    • border-radius

      • Permet d'arrondir les coins d'une image ou bordure

    • border

      • Permet de créer une bordure autour d'un élément
